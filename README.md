## Description

A [Nest](https://github.com/nestjs/nest) API that, given a Steam API Key and a Steam UserID, returns user's owned games and some other information, such as time played and others.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
